## 실행 체크리스트

### 이번 주 (3월 1주)

- [ ] 선배님 피드백을 반영한 로드맵 엑셀 정리 및 공유
- [ ] 오토인코더 아키텍처 초안 설계 (레이어 수, 채널 수 결정)
- [ ] 학습 데이터 전처리 파이프라인 점검 (리사이즈, 정규화)

### Phase 1 — 2D 공간 보존형 특징 추출 (3월)

- [ ] 오토인코더 학습 코드 구현 (인코더-병목-디코더 구조)
- [ ] 재구성 손실 함수 구현 (픽셀 오차 + 구조적 유사도)
- [ ] 잠재 차원 4종(64/128/256/512) 비교 실험
- [ ] 전체 2,033장 잠재 벡터 추출
- [ ] 비지도 군집화 수행 (k=2, k=4)
- [ ] t-SNE / UMAP 2D 시각화
- [ ] 군집 품질 평가 (Silhouette Score, ARI, NMI)
- [ ] 전처리 조건별 비교 (원본 vs 대비 강화 vs 스펙클 필터링)
- [ ] Whole Image vs ROI Crop 비교
- [ ] Phase 1 결과 정리 및 보고

### Phase 2 — 통계적 질감 특징 추출 (4월)

- [ ] 공간적 질감 통계 특징벡터 추출 스크립트 구현
	- [ ] 다양한 거리/방향 조합 설정
	- [ ] 전체 데이터셋 배치 처리
	- [ ] 추출 결과 CSV 저장
- [ ] 국소 미세 패턴 특징 추출 스크립트 구현
	- [ ] 다중 스케일 패턴 히스토그램 추출
	- [ ] 스케일별 결합 벡터 생성
- [ ] 군집화 비교 실험
	- [ ] 질감 통계 단독
	- [ ] 미세 패턴 단독
	- [ ] 질감 통계 + 미세 패턴 결합
	- [ ] Phase 1 잠재 벡터 + 질감 특징 결합
- [ ] 특징 중요도 분석 (어떤 특징이 군집 분리에 가장 기여하는지)
- [ ] 최적 특징 조합 도출

### Phase 3 — 자기지도 표현학습 (5월)

- [ ] 초음파 특화 데이터 변형 파이프라인 구현
	- [ ] 무작위 자르기
	- [ ] 흐림 처리 (스펙클 노이즈 변동성 반영)
	- [ ] 밝기/대비 변동 (장비 세팅 변동성 반영)
	- [ ] 스펙클 노이즈 주입
	- [ ] 좌우 반전
	- [ ] 변형 전후 시각화 → 임상적 왜곡 여부 수동 검증
- [ ] 대조학습 수행
	- [ ] 학습 파이프라인 구축 (positive/negative pair 구성)
	- [ ] 학습 수행 및 수렴 확인
	- [ ] 학습된 인코더에서 임베딩 벡터 추출
- [ ] 표현 품질 평가
	- [ ] 임베딩 공간 군집화 + 2D 시각화
	- [ ] 선형 분류기 4-class 정확도 측정
- [ ] **Phase 1 / Phase 2 / Phase 3 정량 비교표 작성**

### Phase 4 — Patient-level 교차 검증 (6월)

- [ ] 환자 단위 교차 검증 프레임워크 구축
	- [ ] 환자 ID 기준 그룹 분할 구현
	- [ ] 프레임별 예측 → 환자별 집계 로직 구현
- [ ] Phase 1~3 각각의 피처로 Patient-level 성능 측정
- [ ] 피처 결합 실험
- [ ] 핵심 비교: 기존 Malignant Recall <31% 대비 개선 여부 확인
- [ ] 1분기 최종 보고서 작성

### Parallel Track — 기구축 모델 유지관리 (상시)

- [ ] 간 분할 모델 — 주기적 추론 성능 확인
- [ ] 담낭/용종 탐지 모델 — 새 데이터 적용 시 재검증
- [ ] Multi-branch Fusion — Phase 1~3 피처 통합 시 재활용 준비
- [ ] 프레임별 임상 설명 시스템 — 최종 모델 연동 준비
